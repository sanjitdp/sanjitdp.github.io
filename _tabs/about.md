## Introduction
Hey, I'm Sanjit! I'm currently a third-year at UCLA pursuing a concurrent BS/MA degree in math through the Departmental Scholars program, along with a minor in data science engineering. I love applied math and computer science - more specifically, I'm interested in high-dimensional probability, optimization, and some topics in machine learning. If I come across something I find interesting, I'll post it on this site.

If you need to contact me, please use one of the links in the sidebar. Feel free to send me a message or leave a comment if you find any errors.

## Why does this site exist?
Well, as a mathematician, I write *a lot*. I maintain a private repository on GitHub containing project write-ups, solutions to math problems, and code I've written -- I feel it would be nice to consolidate some of the cooler things I've written on this website. I'm a big believer in the philosophy of open source and collaboration, so I'd like to put a lot of this content publicly online. Plus, I think writing some of my current research up for a more general audience than just myself might help me consolidate my understanding of certain topics :)

## Nerdy stuff
This site is written in Jekyll and Ruby using a modification of the `chirpy` theme and deployed using GitHub Pages. I chose to use Jekyll because it's open source, highly customizable, and makes it easy to write (all posts are written in a Markdown flavor called Kramdown). The math is all rendered using MathJax, a popular framework for LaTeX typesetting on the internet. Comments use the `giscus` package, which automatically converts reactions and comments to discussions in the associated [repository](https://github.com/sanjitdp/sanjitdp.github.io) on GitHub. Email subscription is handled by Mailchimp, and emails are sent out automatically when a change to the XML feed is detected through Feedburner. User engagement and page views are tracked using Google Analytics.
